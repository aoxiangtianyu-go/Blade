# Blade Mod（刀锋）

一个用于 **Slay the Spire（杀戮尖塔）** 的简单 Mod，  
仅新增 **一个遗物（Relic）**，围绕「姿态（Stance）」机制进行取舍设计。

---

## ✨ Mod 内容

本 Mod 添加了 1 个新遗物：

### **刀锋（Blade）**

**效果：**
- 在 **愤怒（Wrath）姿态** 下不再受到双倍伤害  
- 你将 **无法改变姿态**
---

## 📦 包含内容

- 1 个遗物
- 2 个能力（Power）
  - **Awe（敬畏）**：愤怒姿态下不再受到双倍伤害
  - **Persistence（恒久）**：阻止姿态切换
- 英文 / 中文本地化

---

## 🛠 运行环境 / 依赖

- **ModTheSpire**
- **BaseMod**

开发时所需的第三方依赖放置在 `lib/` 目录下。
